### Installing :wrench:

Clone this repository:
```bash
git clone https://github.com/franciscojsc/workspace-ubuntu.git
```

Enter the folder:
```bash
cd workspace-ubuntu/src/
```

Assign execute permission to scripts:
```bash
chmod -vR +x .
```

Run the script:
```bash
./install.sh
```
