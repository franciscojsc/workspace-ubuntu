### Installing :wrench:

Clone this repository:
```bash
git clone https://github.com/franciscojsc/workspace-xubuntu.git
```

Enter the folder:
```bash
cd workspace-xubuntu/src/
```

Assign execute permission to scripts:
```bash
sudo chmod +x *.sh
```

Run the script:
```bash
./install.sh
```
